# Probability & Statistics for Data Science — USP

Notes and code from the [USP/ICMC extension course](https://icmc.usp.br) taught by Prof. Francisco Rodrigues.

**Period:** April 6 – May 15, 2026 · 30h recorded + 30h live · 100% online

---

## Topics

1. **Probability Foundations** — Bayes' theorem, random variables, discrete/continuous models, Monte Carlo simulation, LLN, CLT
2. **Statistical Inference & EDA** — Confidence intervals, bootstrap, hypothesis testing, p-values, Bayesian inference
3. **Statistical Modeling & Supervised Learning** — Regression, classification, regularization, logistic regression, statistical learning theory
